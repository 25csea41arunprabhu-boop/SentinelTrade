# 🛡️ SentinelTrade — AI-Driven FinTech Safeguard Dashboard

SentinelTrade is an AI-powered financial risk mitigation system built for **StoxraHack 2026**. It monitors global live crisis feeds (news and emergency alerts) to calculate a real-time **Crisis Volatility Index**, automatically triggering a financial circuit breaker to lock simulated retail portfolios before black swan events crash the market.

## ✨ Key Features
- **Live Crisis Monitoring:** Tracks real-time emergency indicators like cyberattacks, natural disasters, and geopolitical issues.
- **AI Sentiment Analysis:** Uses Natural Language Processing (NLP) to convert news emergency weights into an actionable Risk Score.
- **Automated Circuit Breaker:** Instantly freezes trading postures and suggests defensive hedging (like Gold) when risk crosses a set threshold.
- **Crisis Headline Injector:** A developer mode to inject mock black swan events and test immediate system responses.

## 🛠️ Tech Stack
- **Frontend/Dashboard:** Streamlit
- **AI Core:** NLP Sentiment Analysis
- **Languages:** Pure Python
